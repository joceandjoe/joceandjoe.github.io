---

## ✍️ How to Update
Because this site is managed via the **Publii Desktop App**, do not edit the files directly in GitHub. 

To make changes:
1. Open the **Publii** app on your computer.
2. Edit your pages, posts, or gallery.
3. Click the **"Sync your website"** button.
4. Publii will automatically push the changes to this repository, and the live site will update within a few minutes.
